PT(BR)
Atividade sobre POLIMORFISMO

1)Vamos criar uma classe Conta, que possua um saldo os métodos para pegar saldo, depositar e sacar.

2)Adicione um método na classe Conta, que atualiza essa conta de acordo com uma taxa percentual fornecida.

3)Crie duas subclasses da classe Conta: ContaCorrente e ContaPoupanca. Ambas terão o método atualiza reescrito: A ContaCorrente deve atualizar-se com o dobro da taxa e a ContaPoupanca deve atualizar-se com o triplo da taxa.

Além disso, a ContaCorrente deve reescrever o método deposita, a fim de retirar uma taxa bancária de dez centavos de cada depósito.

4)Crie um formulário para testar o funcionamento de cada uma das classes


EN
POLYMORPHISM activity

1) We will create a class named Conta (Account), which has a saldo <balance> and methods to getSaldo, depositar <deposit> and sacar <withdraw>.

2) Add a method atualiza (update) to class Conta that updates this account with a given tax (rent-roll).

3) Create two subclasses of class Account: ContaCorrente <checking account> and CountaPoupança <savings account>. Both will have the method atualiza overwritten: ContaCorrente must update with double the tax and ContaPoupança must update three times the tax.

Besides that, ContaCorrente must overwrite the method deposita, to charge a bank tax of 10 cents for each deposit.

4) Create a form to test each class.
